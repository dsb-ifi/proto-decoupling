# Project Website Template

[This repository]((https://dsb-ifi.github.io/project-template/)) updates the [jekyllized version](https://github.com/shunzh/project_website) of the source code for the [Nerfies website](https://nerfies.github.io).
You only need to change the content of [index.md](/index.md) in the [gh-pages branch](/dsb-ifi/project-template/tree/gh-pages). 

This branch is the default one where you should place your code.  See the [gh-pages branch](/dsb-ifi/project-template/tree/gh-pages) for details on how to update it.
